# Governance and Safety

Autonomous agents are powerful only when bounded correctly. This experiment uses **constrained autonomy**:

- Agents can analyze, suggest, and execute within approved scopes
- High-impact actions are gated by human approval
- State-changing actions are logged with timestamps and actors
- Evidence is required for actions that affect rewards or status

This structure is not a compliance afterthought. It is core to the experiment. If an adaptive system cannot remain accountable, it is not production-grade storytelling infrastructure.
