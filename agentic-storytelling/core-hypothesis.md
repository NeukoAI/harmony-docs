# Core Hypothesis

Our central hypothesis is that campaigns can become more engaging and more trustworthy when autonomous agents and humans collaborate as a network — not as replacements for each other.

We are testing three claims:

### 1. Responsiveness

A state-aware agent layer can help narrative operations adapt in the same cycle as behavior shifts.

### 2. Relational Intelligence

A campaign that models peer-to-peer relationships can make better participation decisions than one that tracks only isolated user actions.

### 3. Trusted Execution

Agent-assisted operations with human approval checkpoints can reduce lag without sacrificing fairness.
