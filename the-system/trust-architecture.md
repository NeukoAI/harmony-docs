# Trust Architecture

Trust is not a slogan; it is an operational outcome. In this experiment, trust is built through five design choices:

### 1. Legibility

Participants can understand campaign state and progression.

### 2. Consistency

Similar actions are evaluated with similar standards.

### 3. Provenance

High-impact decisions are linked to evidence and review paths.

### 4. Human Sovereignty

Sensitive actions remain approval-gated.

### 5. Feedback Loops

Participants can see that behavior influences narrative and recognition outcomes.

---

This matters because responsiveness without trust feels manipulative, while trust without responsiveness feels inert. The experiment aims to deliver both.
